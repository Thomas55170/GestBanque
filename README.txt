Pour pouvoir lancé le projet GestBanque, ils faut suivre ces différentes étapes:

1. Installer un serveur glassfish
2. Cloner le projet depuis github dans le domain du serveur glassfish (https://github.com/Thomas55170/GestBanque.git)
3. Installer Java EE avec les paramètres classique à tout autre projet.
4. Dans le dossier "conf" de GlassFish copié les fichiers XML se trouvant dans le répertoire "Data" du projet.


Voilà le projet est prêt à être lancé !!

La seule partie manquant est la commande d'un coffre.
